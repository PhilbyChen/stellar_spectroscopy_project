我的第一个天体物理学项目！使用Python分析织女星(vega，αLyr)的光谱数据并尝试确定其光谱型。

实验步骤
数据预处理：读取FITS光谱数据并进行归一化处理
连续谱分析：通过黑体辐射拟合估计有效温度，检测巴尔末跳跃
谱线测量：计算氢巴尔末线系的等值宽度
特征诊断：检测压力展宽
综合判断：结合多重证据确定光谱分类

实验结果：
巴尔末线等值宽度：Hγ = 10.73 Å
巴尔末跳跃位置：3844.7 Å
黑体拟合温度: 12894 K
'等值宽度,'Hα: 6.7029 Å
'等值宽度,'Hβ: 9.9047 Å
'等值宽度,'Hγ: 10.7310 Å
'等值宽度,'Hδ: 10.5598 Å
*压力展宽结果未能提供有效证据，氢线与中性金属线半高全宽差距过小，属巨星范围

基于可靠的光谱特征，织女星的光谱型确定为：
A0V
根据氢巴尔末线强度（Hγ等值宽度10.73Å）判断，符合A型星的典型范围
温度和巴尔末线峰值强度判断，属于A型中最热的亚型
根据显著的巴尔末跳跃和织女星已知的物理参数判断为主序星
